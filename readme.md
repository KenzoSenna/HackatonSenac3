 # Sistema de Prontuário de Enfermagem (Hackaton 2026-1)
  Sistema de Coleta de Dados do Paciente – Técnico em Enfermagem

#  Descrição
Este projeto é um site desenvolvido para auxiliar técnicos e estudantes de enfermagem na coleta e organização de informações de pacientes durante atendimentos e triagens.

#  Objetivo
Facilitar o registro de dados pessoais, sinais vitais, histórico médico e revisão dos sistemas, garantindo um atendimento mais rápido, organizado e seguro.

#  Público-Alvo
- Técnicos em Enfermagem
- Estudantes de cursos técnicos de Enfermagem
- Instituições de ensino e saúde

#  Funcionalidades
- Cadastro completo do paciente
- Registro de sinais vitais
- Histórico médico e familiar
- Controle de medicamentos e alergias
- Revisão dos sistemas
- Campo livre para observações

#  Benefícios
- Agilidade no atendimento
- Padronização das informações
- Redução de erros no preenchimento
- Apoio ao aprendizado de estudantes
- Melhor organização dos dados do paciente

#  Como utilizar
1. Acesse o site
2. Preencha os dados do paciente
3. Registre os sinais vitais e histórico médico
4. Salve ou gere um relatório
